About Project

Project Overview

The Wi-Fi Evil Twin Detection System is a cybersecurity solution designed to identify rogue wireless access points that imitate legitimate Wi-Fi networks. The system analyzes SSID names, BSSID addresses, and signal strength to detect suspicious networks and alert users before they connect.

Objectives

  • Detect duplicate SSIDs.
  • Verify trusted BSSID addresses.
  • Identify potential Evil Twin attacks.
  • Generate security alerts.
  • Maintain scan logs and reports.

Technologies Used

  • Python
  • Flask
  • HTML5
  • CSS3
  • JavaScript
  • SQLite Database
  • PyWiFi
  • Chart.js

How Detection Works

The system scans nearby Wi-Fi networks and compares their SSIDs and BSSIDs with trusted network information. If the same SSID appears with a different BSSID, the network is marked as suspicious and an alert is generated.
